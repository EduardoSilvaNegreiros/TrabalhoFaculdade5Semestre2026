# Entrega 2 - Design e UX

Pasta oficial da segunda entrega do Projeto Integrador, correspondente ao **Bloco 2 - Design e UX** do Beauty Marketplace.

## Objetivo da pasta

Reunir os artefatos que documentam a pesquisa de usuarios, as personas, a jornada, os pontos de dor, os wireframes, o prototipo em Figma e os testes de usabilidade que orientaram a experiencia do sistema.

## Conteudo principal

- `relatorio-entrega-2.pdf`: versao final para submissao.
- `relatorio-entrega-2.md`: fonte textual editavel do relatorio.
- `checklist-avaliacao.md`: conferencia dos blocos esperados para a entrega.

## Leitura recomendada

1. Comece por `relatorio-entrega-2.pdf` para a visao consolidada.
2. Revise `checklist-avaliacao.md` para conferir a cobertura dos itens de UX.
3. Use o link do Figma registrado no relatorio para abrir o prototipo de alta fidelidade.

## Observacao

O relatorio desta entrega foi consolidado a partir do PDF original do grupo e do prototipo em Figma, com normalizacao para o nome atual do projeto, **Beauty Marketplace**, e para a composicao oficial do **Grupo 4**.
