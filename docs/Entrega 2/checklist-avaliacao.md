# Checklist de Avaliacao - Entrega 2

Use esta lista para conferir se a entrega cobre os pontos centrais de Design e UX registrados pelo grupo.

## 1. Identificacao do grupo

- [x] Numero do grupo preenchido.
- [x] Nome completo e RA de todos os integrantes preenchidos.

## 2. Personas

- [x] Persona da consumidora final registrada.
- [x] Persona do lojista parceiro registrada.
- [x] Necessidades, dores e objetivos descritos de forma coerente.

## 3. Jornada do usuario

- [x] Jornada da consumidora organizada por fases.
- [x] Jornada do lojista organizada por fases.
- [x] Cada fase relaciona acao, ponto de dor e oportunidade.

## 4. Pontos de dor e oportunidades

- [x] Logistica multi-vendedor documentada como risco de experiencia.
- [x] Concorrencia e custo de aquisicao tratados como desafio para lojistas.
- [x] Duvida tecnica no momento da compra conectada a suporte inteligente.

## 5. Wireframes e prototipo

- [x] Wireframes de baixa fidelidade registrados como etapa de descoberta e estrutura.
- [x] Link oficial do Figma informado no relatorio.
- [x] Relacao entre prototipo e implementacao final descrita.

## 6. Teste de usabilidade

- [x] Feedbacks consolidados no relatorio.
- [x] Ajustes realizados descritos de forma objetiva.
- [x] Aprendizados de UX conectados a decisoes do projeto.
