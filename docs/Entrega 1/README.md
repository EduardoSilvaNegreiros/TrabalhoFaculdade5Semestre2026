# Entrega 1 - Concepcao

Pasta oficial da primeira entrega do Projeto Integrador, correspondente ao **Bloco 1 - Concepcao** do Beauty Marketplace.

## Objetivo da pasta

Reunir os artefatos que apresentam a definicao inicial do problema, o modelo de negocio, os requisitos do sistema, as user stories, o Kanban e a configuracao do repositorio GitHub.

## Conteudo principal

- `relatorio-entrega-1.pdf`: versao final para submissao.
- `relatorio-entrega-1.md`: fonte textual editavel do relatorio.
- `checklist-avaliacao.md`: conferencia dos requisitos obrigatorios da entrega.

## Leitura recomendada

1. Comece por `relatorio-entrega-1.pdf` para a visao consolidada.
2. Revise `checklist-avaliacao.md` para confirmar o que ja esta completo.
3. Abra o `README.md` da raiz do repositorio para a parte de GitHub e instrucoes de uso.

## Observacao

O relatorio desta entrega foi consolidado a partir do material inicial enviado pelo grupo e complementado com as evidencias atuais do repositorio.
